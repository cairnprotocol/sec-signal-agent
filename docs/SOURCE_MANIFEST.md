# Source Manifest

This repository uses a hand-curated local corpus of short, pre-parsed public EDGAR excerpts. The prototype does not implement a full SEC HTML/XBRL parser; the checked-in excerpts are intentionally small so the demo can focus on signal extraction, deterministic scoring/routing, evidence validation, review queue generation, and audit trace.

## Provenance Model

- Filing excerpts and filing metadata are sourced from public EDGAR records.
- System outputs are generated by the prototype: trigger classification, priority score, workflow route, evidence packet, and audit trace.
- Relationship and review-context metadata is synthetic: review owners, relationship status, strategic relevance notes, and queue examples exist only to demonstrate how a public filing signal would land in a human-owned operating workflow.

All active corpus rows use:

- `source_type = public_edgar_preparsed_excerpt`
- `verification_status = verified_public_edgar_excerpt`

## Corpus Segments

| Segment | Purpose |
| --- | --- |
| `healthcare_edgar_core` | Primary public demo corpus for healthcare commercial, partnerships, operations, trust/security, and compliance review. |
| `cross_industry_edgar_examples` | Verified public EDGAR examples showing that the architecture generalizes beyond healthcare. |
